📊 Sistema de Monitoreo en Python
🧩 Descripción
Proyecto académico y profesional que implementa un sistema modular en Python para monitorear y clasificar datos en distintos sectores:

Inventario de ropa (Grupo Uribe)

Transporte público (Metro de Medellín)

Consumo de agua comunitario (EPM)

Cadena de frío en alimentos (Plaza Minorista)

Cada módulo aplica reglas de negocio específicas y está diseñado con arquitectura escalable, pruebas unitarias y buenas prácticas de programación.

🏗️ Estructura del proyecto
Código
parcial_python/
│
├── main.py                  # Script maestro con menú interactivo
├── grupo_uribe.py           # Control de rotación de inventario
├── metro_medellin.py        # Monitoreo de tiempos de espera
├── epm_consumo.py           # Control de consumo de agua
├── plaza_minorista.py       # Gestión de cadena de frío
├── test.py                  # Pruebas unitarias con unittest
└── README.md                # Documentación del proyecto
⚙️ Tecnologías utilizadas
Python 3.10+

Visual Studio Code

unittest para pruebas automáticas

Métodos de listas (pop, index)

Validaciones de login con intentos limitados

🚀 Instalación y ejecución
Clona el repositorio:

bash
git clone https://github.com/TU_USUARIO/parcial_python.git
cd parcial_python
Ejecuta el sistema:

bash
python main.py
Corre las pruebas unitarias:

bash
python -m unittest discover
🧪 Funcionalidades principales
Login seguro con intentos limitados.

Cálculo de promedios de mediciones.

Clasificación automática según reglas de negocio.

Gestión de listas para manipular datos en tiempo real.
